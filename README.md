# 🛒 Retail Q&A Assistant

Domain-constrained Q&A system that validates questions before answering.

## Features

* ✅ Validates questions are about retail analytics
* ❌ Rejects out-of-scope questions with clear guidance
* 🎯 Keyword-based domain enforcement
* 💬 Interactive demo interface
